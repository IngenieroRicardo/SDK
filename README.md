# SDK
Kit de Desarrollo de Software (SDK) escrito en lenguaje Golang, diseñado para facilitar el desarrollo de aplicaciones en C.

### 🛠️ Compilar SDK

```bash
git clone https://github.com/IngenieroRicardo/SDK.git
cd SDK
make sdk
```

### 🛠️ Compilar main.c

```bash
make compile
```

### 🚀 Ejecutar main.bin

```bash
cd bin
./main.bin
```
